# Desafio Ag Sistemas Back

Desenvolvido com Nest js

### Configurando o projeto

Para configurar o projeto primeiramente devemos ter o node instalado: <br>
https://nodejs.org/en/download <br>

Na raiz do projeto podemos rodar os comandos:

```shell
npm install
```

E para servir:

```shell
npm run start
```

O banco de dados escolhido foi o sqlite para facilitar o desenvolvimento do desafio, logo, não será necessário fazer qualquer configuração de banco de dados.
